## Описание

**SATCOM Трекер** — приложение для наблюдения за положением спутников SATCOM в реальном времени. Показывает азимут и угол места (элевацию), компас для наведения антенны и карту для выбора точки наблюдения.

Данные о положении спутников получаются через [N2YO API](https://www.n2yo.com/api/). Для работы нужен бесплатный API-ключ (регистрация на сайте N2YO).

## Возможности

- **Позиции спутников** — азимут, элевация, расстояние, время обновления
- **Список спутников** — добавление по NORAD ID или названию (с подсказками из загруженной GitHub-базы)
- **RX/TX библиотека** — отображение частот в карточке спутника, экспорт/импорт TXT
- **Импорт TXT с выбором режима**:
  - дополнить текущие данные (без дублей);
  - полностью заменить библиотеку
- **Синхронизация библиотеки с GitHub**:
  - если библиотека пуста — полная загрузка данных без сравнения версий;
  - если библиотека не пуста — проверка версии и обновление только при наличии более новой;
  - перед обновлением показывается предупреждение о перезаписи библиотеки и рекомендация сделать бэкап TXT
- **Работа с каналами в карточке спутника**:
  - добавление пользовательских частот;
  - редактирование `RX/TX/разнос/ширина`;
  - автоматический расчёт разноса при ручном вводе `RX/TX`;
  - комментарии и статус `Offline`;
  - фильтр «Только рабочие каналы»;
  - безвозвратное удаление канала
- **Управление данными частот** — полное удаление библиотеки и пользовательских данных через отдельное окно подтверждения со свайпом
- **Компас** — направление на спутник с учётом магнитного склонения
- **Адаптивный интерфейс iPhone/iPad** — корректная работа в portrait и landscape режимах
- **iPad full-screen режим карточки спутника** — компас и RX/TX библиотека отображаются одновременно в одном экране
- **Источники местоположения** — GPS, ручной ввод координат, выбор точки на карте
- **Карта выбора позиции** — полноэкранная карта, поиск и управляющие кнопки размещены поверх карты
- **Статусы GPS** — подсветка состояния GPS и ошибок с комментариями:
  - в окне выбора точки на карте;
  - в настройках при источнике местоположения `GPS`
- **Темы интерфейса** — `Как в телефоне`, `Светлая`, `Темная`
- **Обновлённый стиль** — новый App Icon в SATCOM-стилистике и чистый градиентный фон основного окна в той же палитре
- **Режимы обновления спутников**:
  - `Автоматически` (интервал 5 мин — 4 ч + ручное обновление);
  - `Отключено` (только данные из кэша, без сетевых запросов)
- **Интерфейс на русском языке**

## Установка и сборка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/DenisSmirnov-ios/SatComTracker.git
   cd SatComTracker
   ```

2. Откройте проект в Xcode:
   ```bash
   open SatComTracker.xcodeproj
   ```

3. Выберите целевое устройство или симулятор и нажмите **Run** (⌘R).

4. При первом запуске откройте **Настройки** (иконка шестерёнки), введите API-ключ N2YO, выберите источник местоположения. Для загрузки частот и базовых спутников используйте «Обновить библиотеку с GitHub».

## API и лимиты

Приложение использует [N2YO REST API](https://www.n2yo.com/api/). Бесплатный ключ имеет ограничения по количеству запросов в день. В режиме `Автоматически` можно задать интервал обновления (рекомендуется не чаще 15–30 минут при большом числе спутников). В режиме `Отключено` сетевые запросы не выполняются, используются только данные из кэша. Запросы к API выполняются батчами (не более 5 одновременно), чтобы не превышать лимиты.

## GitHub-библиотека частот

По умолчанию приложение загружает базу частот из GitHub (`database.csv`) и использует `version.json` для проверки свежести.

Формат `database.csv`:

```csv
rx;tx;name;norad_id
243.990;317.090;125.0°W FltSatCom F8;20253
```

Пример `version.json`:

```json
{
  "version": "2026-03-02T15:45:35Z"
}
```
